AI Coursework - Sarah Herzog

This webpage is an application demonstrating a fuzzy logic controller via a simple racing game. From this page, the user can edit the fuzzy logic parameters and see their results in the game. The user can choose to run the AI through a set track, or to manually control the track during the game's opperation. After the run, the game will display the statistics from it's previous run.

Additionally, a controller can be automatically created using a genetic algorithm optimisation. This controller can then be exported and imported into the main game applicaiton. See the specific instructions in the sections below for more information.

Controller Import / Export

[Back to the top]

This section allows configured controllers to be saved or loaded. This is done via a plain text stream, so the user needs to save this text somewhere of his or her own choice.


Game

[Back to the top]

Control the track manually using the arrow keys. Left moves the track left, right moves it right. Up and down increase or decrease the speed of the track movement. You can also use 1, 2, and 3 to set track movment speed.


Results

[Back to the top]

Once the track above has completed, you can view the results here. The export button will display the results as a stream of plain text, which can be entered into the user's data processing program of choice.



Genetic Algorithm Tuning

[Back to the top]

Uses a genetic algorithm to tune the controller. Run until you get a result within desired range, then export from here and import into the main controller above.
